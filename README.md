# Расписания ВолгГТУ

Интерактивная визуализация расписаний для ВолгГТУ с целью заменить скачивание Excel-файлов

**Первый запуск проекта:** через `docker compose up -d --build` с предварительной настройкой окружения из `.env.example` в `.env` (если запускаете в Docker), либо в `.env.local`, если локально
